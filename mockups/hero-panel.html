Hero Stat Panel

Opened by left-clicking a hero (or approach + interact). Header: name as title, epithet as subtitle, class icon circle + level shield banner top-left, gold bag circle + wealth top-right. Stats: row 1 = HP / Power / Morale, row 2 = HP Regen / Morale Regen, then the rest (gold removed from the list). Labeled inventory separator, then current activity; hireable heroes add centered class info + hire buttons.

Inventory is always exactly three rows — Weapon, Armor, Jewelry — because everyone is wearing at least tier 0 in all three slots. Each slot is sold by its own building (weapon shop / armor shop / jewelry shop). There are no consumables and no fourth slot. Item names are per class per slot per tier (these two heroes are showing tier 0 in everything). Jewelry type is fixed per class: Knight wears a belt, Berserker and Rogue wear rings, Captain/Mage/Cleric wear amulets. Evasion and Accuracy percentages are derived by pitting the hero's raw rating against a reference grunt from the dungeon level they'd be sent to; see the display formula in LordofHirelings_BalanceNumbers.md. A level 1 Mage really does read 0% evasion, and accuracy really is pegged at 100% this early.
