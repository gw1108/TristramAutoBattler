Hero Stat Panel

Opened by left-clicking a hero (or approach + interact). Header: name as title, epithet as subtitle, class icon circle + level shield banner top-left, gold bag circle + wealth top-right. Stats: row 1 = HP / Power / Morale, row 2 = HP Regen / Morale Regen, then the rest (gold removed from the list). Labeled inventory separator, then current activity; hireable heroes add centered class info + hire buttons.
